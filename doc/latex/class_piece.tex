\doxysubsection{Detailed Description}
Tetris pieces description. 

Represents a Tetris piece with blocks, type, pivot, and rotation logic.


\begin{DoxyItemize}
\item Initialize blocks for a given piece type at specific pivot coordinates.
\item Provide access to block positions, pivot, and color.
\item Support rotation around the pivot (clockwise and counterclockwise).
\item Support basic movement operations along the grid. 
\end{DoxyItemize}
